# Personal Portfolio — Quick Start

Open `index.html` in a browser to view the site locally.

To customize:
- Replace the text and links in `index.html` with your name, bio, projects, and social links.
- Add images into an `assets/` folder and update the `src` attributes.
- For contact form submissions, plug in a server endpoint or use a service (Formspree, Netlify Forms).

Deploy suggestions:
- GitHub Pages: push this folder to a repo and enable Pages.
- Vercel / Netlify: connect the repo and deploy (auto-build not required for static sites).
